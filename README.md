# React Native + Express TypeScript App

## Project Structure
- `mobile/` - React Native (Expo) TypeScript frontend
- `backend/` - Node.js Express TypeScript backend

## Setup

1. Install dependencies:
```
npm run install-all
```

2. Run backend (in one terminal):
```
npm run backend
```

3. Run mobile app (in another terminal):
```
npm run mobile
```

## Backend
- Express server running on port 3000
- TypeScript with hot reload via ts-node-dev
- CORS enabled for mobile app communication

## Mobile
- React Native with Expo
- TypeScript configured
- Can run on iOS, Android, or web
