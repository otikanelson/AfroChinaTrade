# E-commerce Enhancement Requirements

## Overview
Comprehensive enhancement of the AfroChina Trade mobile app to include full e-commerce functionality, improved seller tools, and enhanced user experience.

## Customer Features

### 1. Wishlist/Save Products
- Add wishlist functionality to save favorite products
- Display wishlist in customer settings/account page
- Persistent storage in database
- Easy add/remove from product detail pages

### 2. Shopping Cart & Purchase Workflow
- **Add to Cart**: Side panel cart with item count
- **Buy Now**: Direct checkout flow
- **Checkout Screen**: Complete checkout with delivery and payment options
- **Payment Methods**: Store and manage payment methods per user
- **Delivery Addresses**: Store and manage delivery addresses per user
- **Order History**: Complete order tracking and history

## Seller Features

### 3. Enhanced Product Creation
- **Tabular Product Specs**: Structured specification input
- **Policies**: Optional payment, shipping, refund policies
- **Guidelines**: Seller suggestions and best practices

## App Enhancements

### 4. Support & Help System
- **Help & Support Page**: Comprehensive help system
- **Customer Reports**: Issue reporting system
- **App Reviews**: In-app review and feedback system

### 5. Theme & UI Improvements
- **Theme System**: Consistent design language
- **Dark/Light Mode**: Complete theme switching
- **Profile Management**: Picture upload and profile editing

## Technical Requirements

### Database Schema Extensions
- Wishlist collection
- Cart collection (persistent)
- PaymentMethod collection
- DeliveryAddress collection
- Enhanced Order collection
- ProductSpec collection
- HelpTicket collection
- AppReview collection

### API Endpoints
- Wishlist CRUD operations
- Cart management
- Checkout process
- Payment method management
- Address management
- Order processing
- Support ticket system

### Mobile App Components
- Cart sidebar component
- Checkout flow screens
- Payment method forms
- Address management forms
- Wishlist display
- Theme provider
- Profile management screens
