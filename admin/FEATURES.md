# Admin Dashboard Features

## 📊 Dashboard

The main dashboard provides an at-a-glance overview of your business metrics.

### Key Metrics
- **Total Orders**: Number of orders placed
- **Total Revenue**: Sum of all order values
- **Total Users**: Number of registered users
- **Total Products**: Number of products in catalog

### Charts & Trends
- Orders trend over time
- Revenue trend over time
- Recent activity feed

---

## 📦 Products Management

Manage your entire product catalog from one place.

### Features
- ✅ View all products with images
- ✅ Search products by name
- ✅ Filter by category
- ✅ Add new products
- ✅ Edit product details (name, description, price, images)
- ✅ Manage product status (active/inactive/draft)
- ✅ Track inventory/stock levels
- ✅ Delete products
- ✅ Bulk operations (future)

### Product Information
- Product name and description
- Category and subcategory
- Price and stock quantity
- Product images
- Status indicator
- Creation and update dates

Track and manage all customer orders.

### Features
- ✅ View all orders with details
- ✅ Search orders by order number

- ✅ View order items and quantities
- ✅ Update order status
- ✅ Track shipments
- ✅ View customer information
- ✅ Calculate order totals
- ✅ Export orders (future)

### Order Statuses
- **Pending**: Order received, awaiting processing
- **Processing**: Order being prepared
- **Shipped**: Order sent to customer
- **Delivered**: Order received by customer
- **Cancelled**: Order cancelled
