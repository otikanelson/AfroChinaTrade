## What's Ready to Show RIGHT NOW

A professional mobile e-commerce app for Africa-China trade in Nigeria, built with React Native. The foundation is complete with realistic data and a polished interface.

---

## How to Run the Demo

**Simple 3-Step Process:**

1. **Install** (first time only):
   ```bash
   npm install
   ```

2. **Start the app**:
   ```bash
   cd mobile
   npm start
   ```

3. **View on your device**:
   - Press **`w`** for web browser (instant preview)
   - Press **`a`** for Android emulator
   - Press **`i`** for iOS simulator (Mac only)
   - Scan QR code with Expo Go app on your phone

**That's it!** The app launches in under 30 seconds.
