# Assets Directory

## How to Add Your Logo

1. Save your AfroChinaTrade logo image as `logo.png` (PNG format)
2. Copy it to this directory (`mobile/assets/`)
3. Uncomment the Image component in `mobile/src/screens/HomeScreen.tsx`
4. Restart the Expo dev server

The logo should be:
- Format: PNG with transparent background
- Recommended size: 512x512 pixels or larger
- The app will resize it to 80x80 pixels

## Other Assets

You can also add:
- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1284x2778)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)
